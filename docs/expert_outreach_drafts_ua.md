# Готові чернетки першої хвилі експертного outreach

Версія: 1.0  
Дата: 2026-06-28  
Статус: **draft only — нічого не надіслано**

Ці листи поєднують загальний шаблон із `validation_launch_kit_ua.md` та перевірений
shortlist E01–E05. Перед відправленням заповніть поля у квадратних дужках і повторно
перевірте контакт на офіційній сторінці.

Обов’язкові поля:

- `[ІМ’Я ВІДПРАВНИКА]`;
- `[РОЛЬ / КОМАНДА]`;
- `[КОНТАКТ]`;
- `[ФОРМАТ КОМПЕНСАЦІЇ]` або фраза «готові обговорити компенсацію»;
- `[БАЖАНИЙ ПЕРІОД]` без ультимативного дедлайну;
- `[ПОСИЛАННЯ НА ПАПКУ / GITHUB]`.

## Спільний пакет посилань

У кожному листі дайте одну стартову папку або ці чотири посилання:

## Одне нейтральне нагадування

Надіслати після погодженого строку або через 5–7 робочих днів. Не надсилати частіше.

**Тема:** Re: [попередня тема]

> Добрий день. Коротко нагадую про запит щодо незалежного рев’ю Milestones.
> Якщо scope завеликий, можемо звузити його до одного домену або віку. Якщо запит
> адресовано не тій людині, буду вдячний/вдячна за правильний контакт. Якщо участь
> зараз неможлива, достатньо короткої відповіді — додатково нагадувати не будемо.
>
> Дякую,  
> [ІМ’Я ВІДПРАВНИКА]

Після другого контакту без відповіді закрийте маршрут як `no response` і переходьте до резерву.

## Checklist перед натисканням Send

- [ ] Контакт повторно перевірений на офіційному сайті.
- [ ] Ім’я та звертання правильні.
- [ ] Вказано реального відправника і зворотний контакт.
- [ ] Scope відповідає професійній ролі адресата.
- [ ] Немає тверджень про «клінічне схвалення» або готовність продукту до користувачів.
- [ ] Формат компенсації не прихований і не обіцяє неузгоджену суму.
- [ ] Бажаний строк сформульований як пропозиція.
- [ ] Посилання відкриваються без доступу до приватних даних.
- [ ] У вкладеннях немає книг, source PDFs, даних дітей або всього репозиторію.
- [ ] У tracker підготовлено рядок для статусу контакту.
- [ ] Відправлення прямо дозволене власником продукту.
