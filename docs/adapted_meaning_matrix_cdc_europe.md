# Адаптована матриця сенсів для CDC x Europe Milestones

Версія: попередня адаптація під продукт `milestones + activities + evidence`, а не під продаж класичного курсу.

## Головний зсув

Оригінальна матриця добре продає курс через біль, тривогу, провину і бажання "дати дитині максимум". Для продукту `CDC x Europe Milestones` ми залишаємо ці мотивації, але змінюємо тон:

- було: "виправити", "надолужити", "не пропустити критичне вікно";
- стає: "спостерігати розвиток", "підтримувати навички через гру", "розуміти, коли варто поговорити з фахівцем";
- було: авторська система як головний доказ;
- стає: CDC/AAP/WHO як каркас, європейські review windows як рамка, вправи як практична підтримка;
- було: курс як головний продукт;
- стає: age-based milestone map, activity cards, 7-day plan, progress notes, clinician-friendly summary.

## Безпечна рамка продукту

Продукт не має звучати як діагностика, лікування чи гарантія розвитку. Базова фраза:

> Ми допомагаємо батькам бачити, що зазвичай з'являється у цьому віці, підтримувати розвиток через короткі ігрові активності та знати, коли варто обговорити питання з педіатром або спеціалістом.

## Пріоритет сегментів для MVP

1. `Та, хто боїться пропустити` - головна аудиторія першої версії 0-12 months.
2. `Та, хто виснажена` - найкращий сегмент для щоденного usage і retention.
3. `Та, хто на чужині` - стратегічна перевага для Luxembourg/EU.
4. `Та, хто хоче бути найкращою мамою` - premium/evidence позиціонування.
5. `Та, хто сумнівається у вартості` - потрібна для checkout/landing/offer.
6. `Та, хто вже бачить проблему` - важлива, але тільки з дуже обережною мовою і professional referral логікою.

## Messaging Guardrails

Не використовувати як основні claims:

- "виправимо затримку";
- "надолужимо розвиток";
- "гарантований результат";
- "критичне вікно закриється";
- "дитина випереджатиме ровесників";
- "замість логопеда/педіатра/невролога";
- "80% проблем мовлення через...";
- "клінічно доведено", якщо немає конкретного дослідження саме на цю вправу/протокол.

Використовувати:

- "підтримує передумови навички";
- "допомагає спостерігати";
- "може бути корисною частиною щоденної гри";
- "evidence-informed";
- "aligned with CDC/AAP/WHO guidance";
- "якщо вас щось турбує, обговоріть це з фахівцем";
- "monitor / discuss with professional";
- "source-backed, not fear-based".

## Приклад переписаного меседжу

Було:

> Якщо ці 3-6 місяців пройдуть без правильних стимулів - мозок просто не отримає потрібних сигналів.

Краще:

> У перші місяці мозок активно вчиться через повторювану взаємодію з дорослим. Тому короткі щоденні ігри - погляд, голос, рух, дотик - можуть підтримувати навички, які зазвичай з'являються в цьому віці.

Було:

> Курс допоможе виправити затримку мовлення.

Краще:

> Якщо мовлення вас турбує, важливо обговорити це з фахівцем. Паралельно вдома можна підтримувати передумови мовлення: спільну увагу, відповідь на звуки, жести, turn-taking і щоденну взаємодію.
